A robust and secure ATM management system implemented in C, featuring comprehensive account management, transaction handling, and user authentication.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item User authentication (login/registration)
\item Account management
\begin{DoxyItemize}
\item Create new accounts
\item Update account information
\item Check account details
\item Remove accounts
\item Transfer ownership
\end{DoxyItemize}
\item Transaction handling
\begin{DoxyItemize}
\item Deposits
\item Withdrawals
\end{DoxyItemize}
\item Multiple account types
\begin{DoxyItemize}
\item Savings accounts
\item Fixed-\/term accounts (1, 2, and 3 years)
\end{DoxyItemize}
\item Security features
\begin{DoxyItemize}
\item Password protection
\item Failed login attempt handling
\item Data validation
\item File backup system
\end{DoxyItemize}
